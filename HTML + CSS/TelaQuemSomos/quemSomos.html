QUEM SOMOS?

A Moto Fácil é um projeto de software web desenvolvido nas disciplinas: Análise e Projeto de Software e Desenvolvimento de Aplicações Web. O projeto em questão, foi ministrado pela professora Isabelle Lima e realizado pelos alunos Beatriz Gomes e Daniel Costa, do Instituto Federal da Paraíba.


A escolha desse projeto se deu ao encarar um problema comum em cidades do interior: a escassa automatização desses serviços e a difícil comunicação que os passageiros tinham com os motoristas.


OBJETIVO

Nosso objetivo é facilitar a vida dos motoristas e passageiros, automatizando todo o processo, para que você tenha acesso aos melhores serviços.
